Is it possible to use AI without needing to code?

The answer is yes!

Is a data set is enough to build a machine model?

Yeah that's it..!
All you need to build a machine learning model is a data set.

Now we have a lot of code-free tools Azure ML Studio, DataRobot, and deep cognition. here we use a python library called Ludwig that Uber been using internally for 2 years

Recently Uber’s AI lab open sourced their python library, Ludwig

Ludwig is a toolbox built on top of TensorFlow that allows to train and test deep learning models without the need to write code.
It is built with extensibility principles in mind and is based on data type abstractions, making it easy to add support for new data types as well as new model architectures.

A suite of visualization tools allows you to analyze models’ training and test performance and to compare them.

SET UP(Installation)

You will be needing python 3 for installation. If you don't have

install it by running:
sudo apt install python3  # on ubuntu
brew install python3      # on mac

In order to install Ludwig all you need to do is just run the following:

pip install ludwig
python -m spacy download en

Train

Training a model is pretty straightforward: you provide a dataset and a model definition.

The model definition contains a list of input features and output features, all you have to do is specify names of the columns in the CSV that are inputs to your model alongside with their datatypes, and names of columns in the CSV that will be outputs, the target variables which the model will learn to predict. Ludwig will compose a deep learning model accordingly and train it for you.

Prepare your data in a CSV file, define input and output feature in a model definition YAML file and run:

ludwig train
--data_csv file.csv
--model_definition definition.yaml

This will perform a random split of the data, preprocess it, build a model that decodes output classes through a softmax classifier, train the model on the training set until the accuracy on the validation set stops improving. Training progress will be displayed in the console, but TensorBoard can also be used.

Predict

Prepare your data in a CSV file and use a pre-trained model to predict the output targets:

ludwig predict
--data_csv data.csv
--model path_to_model

Running this command will return model predictions and some test performance statistics if the dataset contains ground truth information to compare to.

Those can be visualized by the visualize tool, which can also be used to compare performances and predictions of different models

Visualize

Ludwig comes with many visualization options. If you want to look at the learning curves and of your model, run:

ludwig visualize
--visualization learning_curves
--training_statistics train_statistics.json

Compare performances and predictions of different models using visualize

Train and predict directly from Python

If you want to use this in your python code, you can do it. This also provides a simple programmatic API that allows you to train or load a model and use it to obtain predictions on new data:

from ludwig import Ludwig Model

# train a model
model_definition = {...}
model = LudwigModel(model_definition)
train_stats = model.train(training_dataframe)

# or load a model
model = LudwigModel.load(model_path)

# obtain predictions
predictions = model.predict(test_dataframe)

model.close()

Building a Simple Text Classification Model using Ludwig

With model_definition.yaml:

input_features:
   -
       name: text
       type: text
       encoder: parallel_cnn
       level: wordoutput_features:
   -
       name: class
       type: category

experiment: Runs a full experiment training a model and testing it

ludwig experiment \
 --data_csv data.csv \
 --model_definition_file model_definition.yaml
